Operating Systems Terminology

Term Definition
Operating System An Operating System is the software that controls the overall operation of a computer.
Job Queue A Job Queue is a storage organization in which jobs are ordered first-in first-out.
FIFO FIFO stands for First-In First-Out meaning objects in a queue are removed in the order they arrived.
Interactive Processing a feature developed in operating systems allowing a program to be ecevuted to carry on a dialogue with the user through remote terminals.
Time-Sharing A feature in operating systems that provide service to multiple users at the same time.
Multiprogramming A means of implementing time-sharing by dividing time into two intervals then the execution of a job is restricted to only one interval at a time.
Application Software Application software consists of the programs for performing tasks particular to the machine's utilization.
System Software System software performs tasks that are common to computer systems in general providing the infrastructure that application software requires to operate.
Utility Software Ulility Software functions as a operating system extension consisting of programs that perform activities that are fundermental for a perticular installation.
Graphical User Interface (GUI) An interface where objects like files or programs can be manipulated. said objects are represented pictorally by icons
Shell A shell is the name for older user interfaces that communicated with the user via textual messages on a monitor in in opposed to the modern GUI.
Kernel A kernel is the internal part of an operating system containing the software components that perform the very basic functions required by the computer installation.
Device Driver A device driver is a software unit that communicates with controllers to carry out operations on the peripheral devices attached to a machine.
File Manager The file manager's job is to coordinate the use of the machine's mass storage facilities.
Memory Manager The memory manager's main task is tho coordinate the machine's use of main memory.
Virtual Memory Virtual memory is a large "fictional" memory space created by paging.
Boot Strapping Boot strapping is performed by a computer each time it is turned on by transferring the operating system from mass storage into main memory.
ROM ROM (read-only memory) is a small portion memory stored in special nonvolatile memory cells for the CPU to use to find its initial program.
Deadlock Deadlock is an error that can happen during resource allocation when two or more processes are blocked because each is waiting for a resource that is allocated to another.
Spooling A technique of holding data for output at a later but more convenient time
Super User / Administrator A super user is a person with highly priveleged access to the operating system by identifying him- or herself as the administrator via a login procedure.
Multi-Factor Authentication A means of verifying the authenticity of a user by requiring 2 or more means of authentication (EX. login password + text code verification)
