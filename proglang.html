Data Types:
  • Integer - numerica data consisting of whole numbers, probably stored using two's complement notation.
  • Float - the type float (sometimes called real) refers to numeric data that might contain values other than whole numbers, probably stored in floating-point notation.
  • String - a string is a data type used in programming to represent text. it is a sequence of characters, where each character can be a letter, digit, punctuation mark, whitespace, or any other symbol. in most programming languages, strings are considered a fundamental data type, and they are often used to manipulate and store textual data.
Primitive Date Type the data types that are included as primatives in a programming language, such and "int" for integer or "char" for character.
Data Structure the conceptual shape or arrangement of data.
Python Collection Data Structure
  • List - data structure that is used to store and organize a collection of items. It is a mutable, ordered sequence of elements.
  • Tuple - tuples are similar to list exept a tuple is immutable. once you create a tuple, you cannot modify its contents - you cannot add, remove, or change elements.
  • Set - unlike lists and tuples, a set is an unordered collection of unique elements. this means that a set cannot contain duplicate elements, and the order in which elements are added is not preserved.
  • Dictionary - a dictionary is another important built-in collection data type in python. it is used to store and retrieve key-value pairs. each element in a dictionary consists of a key and its associated value, and the key serves as a unique identifier for that value.
Function a set of instructions for programming a task that can be used as an abstract tool by other program units.
Local Variable a varaible declared within a function.
Global Variable a variable thats scope is not restricted to a specific part of a program.
Source Program a source program or source code is a program consisting of human-readable sets of instructions that a programmer creates to implement a specific algorithm or functionality written in a high-level programming language.
Object Program an object program is the output generated by a compiler or an assembler after processing the source program.
